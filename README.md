# ATAC-seq analysis pipeline

ATAC-seq pipeline wrote in Snakemake. Work in progress ;)

## Prerequisites

I put the versions I'm using in brackets. 

* fastqc (0.11.8)
* multiqc (1.5.dev0)
* cutadapt (2.1)
* trim\_galore (0.5.0)
* bwa (0.7.17)
* samtools (1.7)
* picard (2.19.0)
* conda (4.7.5)
* macs2 (2.1.2)
